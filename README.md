# YouTube Media Extractor

Este script en Bash permite descargar un vídeo de YouTube sin audio en el formato que elijas y extraer el audio en formato MP3.

## Requisitos

- Ubuntu (o Debian)
- Conexión a internet
- Permisos para instalar paquetes (sudo)
